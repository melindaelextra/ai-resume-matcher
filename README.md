# AI Resume Matching System

## Project Overview
The AI Resume Matching System is a project that analyzes resumes and compares them with job descriptions to evaluate candidate-job fit. The system generates a match score, identifies missing skills, and provides suggestions to improve the resume based on job requirements.

This project simulates how companies automatically screen resumes and rank candidates using data processing and natural language processing techniques.

## Features
- Parse and extract text from PDF resumes
- Extract key sections such as skills, education, and experience
- Compare resume content with job description
- Generate match score
- Identify missing skills
- Provide suggestions for resume improvement
- Backend API built with FastAPI

## Technologies Used
- Python
- FastAPI
- PyMuPDF (PDF text extraction)
- Sentence Transformers (semantic similarity)
- Scikit-learn
- Pandas / NumPy

## Project Structure
- Resume Parser
- Job Description Parser
- Matching Engine
- Scoring System
- API Server
